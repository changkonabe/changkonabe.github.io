An Intro to Java.

Java is a language in which its programs are compiled. This differs from Python, which is implemented mainly through interpretation. As you may have seen with Python in CS61A, interpreted means the program is executed line by line as it is run. What it means to be compiled is that a software known as a compiler takes code and translates it to another language, usually one at a lower level and closer to machine language. It is this compiled code that is then executed in order to run the program.

Primitives and Typing

In Java, there are 8 primitive types, which are basic data types implemented and supported by the language. They are as follows:

  • boolean: true or false.
  • char: 16-bit Unicode character such as 'a'.
  • byte: 8-bit two's complement integer.
  • short: 16-bit two's complement integer.
  • int: 32-bit two's complement integer.
  • long: 64-bit two's complement integer.
  • float: single precision 32-bit floating point.
  • double: double precision 64-bit floating point.

Java as a language is statically typed. This means all declaration of variables specifies the type of the variable in that scope. So, if we want to declare a variable to store primitive ints, we could have a declaration like:

int x;
x = 5;

Note that then the line

x = true;

would cause the program to fail at compilation because we see the variable x in this scope is already of static type "int", so it is not possible to redeclare its value to be something of type boolean.
